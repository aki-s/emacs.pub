PEAR_Error::getMode()

PEAR_Error::getMode() – get error mode

Synopsis

require_once 'PEAR.php';

integer PEAR_Error::getMode ( )

Description

Returns the error mode used for throwing the error object.

Return value

integer - error mode

Note

This function can not be called statically.
