Validate_IE

This package is in beta state

Package contains locale validation for Ireland such as:

  1. Bank Account

  2. Drivers Licence Numbers

  3. IBAN

  4. Passport Numbers

  5. Phone Numbers

  6. Postal Code

  7. SSN/PPSN

  8. SWIFT Codes
