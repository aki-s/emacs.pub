Auth::logout()

Auth::logout() – logout an authenticated user

Synopsis

void Auth::logout ( )

Description

Logout any currently logged in user.

Example

<?php
//$a is our Auth object
$a->start();

if (isset($_GET['action']) && $_GET['action'] == 'logout'
    && $a->checkAuth()
) {
    $a->logout();
    $a->start();
}
?>
