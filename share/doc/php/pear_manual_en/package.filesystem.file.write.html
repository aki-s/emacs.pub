File::write()

File::write() – writes bytes to a file

Synopsis

require_once 'File.php';

mixed File::write ( string $filename , string $char , string $mode = FILE_MODE_APPEND , mixed $lock = false )

Description

Parameter

Return value

Throws

Note

See

Example

Using File::write()

<?php
require_once 'File.php';

$e = File::write('test.txt', 'this is a test line', FILE_MODE_WRITE);

if (PEAR::isError($e)) {
    echo 'Could not write to file : ' . $e->getMessage();
} else {
    echo "Successfully wrote to file test.txt\n";
}
?>
