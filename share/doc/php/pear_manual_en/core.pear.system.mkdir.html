System::mkDir – create directories

Synopsis

require_once "System.php";

bool System::mkDir ( string $args )

Description

Creates directories.

Parameter

  • string $args - the name of the directory/-ies to create

Return value

bool - TRUE for success

Note

This function can be called statically.
