Net_FTP::disconnect()

Net_FTP::disconnect() – disconnects from the FTP server

Synopsis

require_once 'Net/FTP.php';

void Net_FTP::disconnect ( )

Description

Disconnect from the FTP server you're connected to.

Return value

void

Note

This function can not be called statically.

Example

Using disconnect()

<?php
     
    $test->disconnect();

?>
