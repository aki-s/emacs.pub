Validate_ptBR

This package is in alpha state

Package contains locale validation for Brazil such as:

  1. Social Inurance Numbers (SIN)

  2. Region (brazilian states)

  3. Postal Codes

  4. Phone Numbers

  5. CNPJ

  6. CPF

  7. Vehicle's plate
