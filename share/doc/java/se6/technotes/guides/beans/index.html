JavaBeansTM Component API
The JavaBeans component API extends the Java platform's Write Once, Run AnywhereTM capability to reusable component development. In fact, the JavaBeans architecture takes interoperability a major step forward â€” your code runs on every OS and also within any application environment. A developer who creates components based on the JavaBeans architecture (beans) secures a future in the emerging network software market without losing customers that use proprietary platforms, because beans interoperate with ActiveX, OpenDoc, and LiveConnect. Long-term persistence for JavaBeans enables you to read and write a bean as a textual representation of its property values.
