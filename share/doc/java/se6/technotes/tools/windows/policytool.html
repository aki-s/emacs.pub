policytool - PolicyTool Administration GUI Utility

policytool reads and writes a plain text policy file based on user input via the utility GUI.
  • Synopsis
  • Description
  • Options
  • See Also

SYNOPSIS

 
Run the policytool Administrator's utility
policytool
Run policytool and load the specified policy file
policytool[-file][filename]
 
where:
file
directs policytool to load a local policy file
filename
The file name

DESCRIPTION

policytool is an Administrator's GUI that allows system administrators to manage the contents of local policy files. A policy file is a plain-text file with a .policy extension, that maps remote requestors by domain, to permission objects. For details, see Default Policy Implementation and Syntax.

OPTIONS

file
Loads filename.
